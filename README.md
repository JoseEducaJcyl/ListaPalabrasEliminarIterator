# Filtro de Palabras con 'A'

## Descripción
Aplicación de consola en Java que elimina de una lista todas las palabras que contengan la letra 'A' (mayúscula o minúscula) utilizando un Iterator.

## Funcionalidades
- Lista predefinida de palabras
- Eliminar palabras que contengan la letra 'A' o 'a'
- Mostrar las palabras restantes

## Tecnologías
- Java
- ArrayList
- Iterator
